## 🔌 API Endpoints

### Аутентифікація
```
POST   /api/auth/register    - Реєстрація
POST   /api/auth/login       - Вхід
```

### Користувачі
```
GET    /api/users/:id        - Отримати профіль
PUT    /api/users/:id        - Оновити профіль
```

### Пости
```
GET    /api/posts            - Получить ленту
POST   /api/posts            - Создать пост
GET    /api/posts/:id        - Получить пост
GET    /api/posts/user/:userId - Получить посты пользователя
DELETE /api/posts/:id        - Удалить пост
```

### Коментарі
```
POST   /api/comments         - Добавить комментарий
GET    /api/comments/post/:postId - Получить комментарии
DELETE /api/comments/:id     - Удалить комментарий
```

### Лайки
```
POST   /api/likes            - Лайк пост
DELETE /api/likes/:postId    - Разлайк пост
GET    /api/likes/post/:postId - Получить лайки
```

### Підписки
```
POST   /api/follows          - Подписаться
DELETE /api/follows/:followingId - Отписаться
GET    /api/follows/followers/:userId - Получить подписчиков
GET    /api/follows/following/:userId - Получить подписки
```

### Повідомлення
```
POST   /api/messages         - Отправить сообщение
GET    /api/messages         - Получить входящие
GET    /api/messages/conversation/:otherUserId - Получить диалог
```

### Пошук
```
GET    /api/search/users?q=query - Поиск пользователей
GET    /api/search/hashtags?q=query - Поиск хештегов
GET    /api/search/hashtags/:name/posts - Посты по хештегу
```

## 🚀 Швидкий Старт

### Локальне тестування

1. **Встановіть Node.js 18+**
2. **Встановіть PostgreSQL**
3. **Клонуйте проект**
```bash
cd c:\Users\vladu\Desktop\Nexus
```

4. **Встановіть залежності**
```bash
npm install
```

5. **Налаштуйте .env**
```
cp .env.example .env
# Відредагуйте .env з вашими параметрами
```

6. **Запустіть сервер**
```bash
npm run dev
```

7. **Відкрийте браузер**
```
http://localhost:5000
```

### Розгортання на Railway

Див. файл `RAILWAY_DEPLOYMENT.md` для детальних інструкцій.

**Коротко:**
1. Push на GitHub
2. Зареєструйтесь на Railway.com
3. Підключіть GitHub репозиторій
4. Додайте PostgreSQL
5. Встановіть змінні (JWT_SECRET тощо)
6. Готово! 🎉
