# Швидкий старт Nexus

## 🚀 Розгортання на Railway за 5 хвилин

### Крок 1: Підготовка GitHub
1. Форкніть або клонуйте цей репозиторій на GitHub
2. Переконайтесь, що файли у папці `Nexus` є у вашому репозиторії

### Крок 2: Реєстрація на Railway
1. Перейдіть на [https://railway.app](https://railway.app)
2. Натисніть "Login" або "Get Started"
3. Виберіть "Continue with GitHub"

### Крок 3: Створення проекту
1. У Railway натисніть "New Project"
2. Виберіть "Deploy from GitHub repo"
3. Авторизуйте Railway на GitHub
4. Виберіть ваш репозиторій

### Крок 4: Додавання PostgreSQL
1. У Railway натисніть "Add Service"
2. Пошукайте "PostgreSQL"
3. Натисніть "PostgreSQL"
4. Виберіть цінову категорію (desenvolvimento є вільно)

### Крок 5: Налаштування змінних
1. Натисніть на ваш проект
2. Перейдіть на вкладку "Variables"
3. Додайте змінні:
   ```
   JWT_SECRET=ваш_супер_таємний_ключ_мінімум_20_символів
   NODE_ENV=production
   PORT=3000
   ```

### Крок 6: Розгортання
1. Натисніть "Deploy"
2. Чекайте завершення (~ 2-3 хвилини)
3. Натисніть на URL вашого застосунку

## 🛠️ Розробка локально

```bash
# Встановіть залежності
npm install

# Запустіть PostgreSQL локально
# createdb nexus_db

# Скопіюйте .env.example в .env та відредагуйте

# Запустіть сервер з HotReload
npm run dev

# Або звичайний запуск
npm start
```

## 📝 Поширені проблеми

### Помилка: "Cannot find module"
```bash
npm install
```

### Помилка: "Connection refused"
Переконайтесь, що PostgreSQL запущений і DATABASE_URL правильний

### Сайт не завантажується
Перевірте логи Railway (натисніть "Logs" в Dashboard)

## 🔧 Додаткова налаштування

### Зміна пускового порту
У файлі `server.js` змініть:
```javascript
const PORT = process.env.PORT || 5000;
```

### Додавання SSL для вашого домену
1. У Railway на вкладці "Domains" натисніть "Add Domain"
2. Введіть ваш домен
3. Слідуйте інструкціям для налаштування DNS
