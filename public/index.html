Welcome to Nexus — an Instagram-like social network built with Node.js, Express, and PostgreSQL. The API is live and endpoints are being implemented.

API Endpoints

  • POST /api/auth/register — Create account
  • POST /api/auth/login — Sign in
  • GET  /api/posts — Browse feed
  • GET  /api/users/:id — View profile
  • GET  /api/search/users — Search users
